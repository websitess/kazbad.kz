Косметическая банка 75 мл., подходит для фасовки косметических масок, кремов, солей для ванн. Баночка изготовлена из ПЭТ материала. Комплектуется резьбовой крышкой. Для герметичности и сохранения свежести, возможна индукционная запайка банки с помощью запечатывания горловины алюминиевым или клеевыми вкладышем. Пластиковая упаковка (банка и крышка) для косметики может быть окрашена в любые цвета.
